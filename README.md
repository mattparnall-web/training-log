# Coach

Personal training, nutrition, and coaching hub with cloud sync via Supabase.

Tabs:
- **Dashboard** — morning briefing pulling from Garmin + training + intake
- **Workout** — strength + cardio session logging (the original tracker)
- **Food** — meal logging (photo + AI parse, manual fallback)
- **Alcohol** — quick-tap drink logger with UK alcohol units
- **Settings** — calorie target, target weights per exercise

Originally named "Training Log" — renamed to "Coach" when expanded into a multi-tab fitness hub with an AI coaching loop.

## Deploy

1. Push to GitHub (auto-deploys to Vercel)
2. Add to iPhone home screen via Safari → Share → Add to Home Screen

## Local dev

```
npm install
npm run dev
```
